# Proyecto-DSS

## Descripción del proyecto

Proyecto-DSS es una aplicación web de salud y fitness personalizado desarrollada con Laravel, PHP y MySQL, orientada a ayudar a los usuarios a gestionar su progreso físico, organizar rutinas de entrenamiento y llevar un control detallado de su evolución corporal.

El sistema permite a los usuarios registrarse e iniciar sesión para acceder a un perfil personalizado donde pueden administrar información relacionada con su condición física, objetivos y progreso. Además, la plataforma incluye funcionalidades para crear y gestionar rutinas de ejercicios, registrar peso corporal y visualizar historiales de entrenamiento.

Como parte de la solución, el proyecto integra APIs externas para consultar información de ejercicios y datos nutricionales, permitiendo ofrecer contenido actualizado sobre entrenamientos, calorías, proteínas y otros valores alimenticios relevantes para el bienestar del usuario.

La aplicación surge como una alternativa a las plataformas actuales de fitness, las cuales suelen presentar limitaciones en personalización, seguimiento del progreso y centralización de funciones. Por ello, este sistema busca brindar una experiencia más organizada, intuitiva y enfocada en las necesidades individuales de cada usuario.

### Funcionalidades principales

- Registro e inicio de sesión de usuarios
- Gestión de perfil de usuario
- Creación y administración de rutinas personalizadas
- Registro y seguimiento del peso corporal
- Historial de entrenamientos
- Integración con APIs de ejercicios y nutrición
- Seguimiento de métricas físicas
- Interfaz amigable y fácil de utilizar

El proyecto fue desarrollado bajo una arquitectura cliente-servidor utilizando tecnologías web modernas y buenas prácticas de desarrollo, con el objetivo de ofrecer una solución eficiente, escalable y funcional para el área de salud y fitness.
